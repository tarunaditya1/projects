      About MAC

      MAC Cosmetics (Make-up Art Cosmetics) is known for professional-grade pigments, expansive shade ranges, and artist‑forward innovation across face, eyes, and lips.

      How to Update the Catalog

      1. Open data/products.js.
      2. Copy an existing product object in the window.PRODUCTS array and edit the fields.
      3. Add shade entries with name and hex (for swatch color). If you don't know the hex, use a neutral like #bdbdbd.
      4. Optional: add an external buyLink to the official product page.
